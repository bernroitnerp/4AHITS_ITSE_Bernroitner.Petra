# Reverse Shells Übungen

## Übung (netcat chat)
Verwende netcat um einen bidirektionalen Chat zwischen 2 unterschiedlichen Rechnern aufzubauen.


Was heißt *bidirektional*?

Ein bidirektionaler Chat bedeutet schlichtweg, dass Daten (in diesem Fall Textnachrichten) in beide Richtungen gleichzeitig fließen können. 
Beide Rechner fungieren sowohl als Sender als auch als Empfänger.


**Man hat 2 Rechner: Rechner A und Rechner B**

**Schritt 1: Rechner A öffnet den Port (Listening)**
Rechner A muss auf eine eingehende Verbindung warten. Wir wählen einen Port (z. B. 4444), der nicht von anderen Programmen belegt ist.

Gib folgenden Befehl ein:
```Bash
nc -l -p 4444
```

-l: Steht für Listen (Zuhören).

-p 4444: Gibt den Port an.

**Schritt 2: Rechner B verbindet sich**
Rechner B muss nun aktiv die Verbindung zu Rechner A aufbauen.

Gib folgenden Befehl ein:

```Bash
nc [IP-Adresse von Rechner A] 4444
```

**Wie man chattet**
Sobald Rechner B die Verbindung hergestellt hat, scheint im Terminal erst einmal nichts zu passieren. Das ist das Zeichen, dass die Leitung steht!

Tippe eine Nachricht auf Rechner A und drücke Enter. Sie erscheint sofort bei Rechner B.
Tippe eine Nachricht auf Rechner B und drücke Enter. Sie erscheint sofort bei Rechner A.

Um den Chat zu beenden, drücke auf einem der Rechner ```Strg + C```.
